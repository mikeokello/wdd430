# Library Management Hub - Project Specification

## 1. Overview
Library Management Hub is a Next.js web application that helps students and librarians manage book inventory, borrowing, returns, and user accounts in one centralized system.

## 2. Target Users
- Students: Browse, search, borrow, and return books
- Librarians: Manage inventory and user accounts

## 3. Core Features
- User Authentication: Sign up, login, logout
- Book Management: CRUD for books
- Borrowing System: Borrow and return books
- Search and Filter: Find books by title, author, genre

## 4. Tech Stack
- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: Next.js API Routes
- Database: PostgreSQL with Prisma ORM
- Deployment: Vercel

## 5. User Stories
- As a student, I want to search for books so I can find what I need
- As a student, I want to borrow a book so I can take it home
- As a librarian, I want to add new books so the inventory is updated

## 6. Acceptance Criteria
MVP must have user auth, book CRUD, borrow/return functionality, and responsive design.
